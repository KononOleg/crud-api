# CRUD-API

### Clone the repo
```bash
git clone https://github.com/KononOleg/crud-api.git
cd crud-api
git checkout develop
```

### Install the dependencies
```bash
npm install
```

### Run in development mode
```bash
npm run start:dev
```

### Run in production mode
```bash
npm run start:prod
```

### Run test
```bash
npm run test
```
